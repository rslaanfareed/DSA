# DSA Labs — C++

Data Structures and Algorithms implementations in C++, covering core concepts studied during my second year at UET Taxila.

## Topics Covered

- **Arrays & Sorting** — Bubble sort, linear search, binary search with repeating elements
- **Linked Lists** — Singly linked list, sorted linked list, doubly linked list, circular linked list (song playlist)
- **Stacks** — Array-based and linked list-based stack, string reversal, parenthesis validation
- **Queues** — Circular queue, priority queue
- **Hashing** — Hash table with linear probing, hash table with chaining
- **Trees** — Binary Search Tree (insert, delete, search, preorder traversal)
- **Infix and Postfix** — Infix to postfix conversion, postfix evaluation
- **Matrix Operations** — Matrix manipulation
- **University Bus Scheduling** — Applied DSA problem

## Language
C++
